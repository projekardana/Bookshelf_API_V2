# Bookshelf API

A simple RESTful API for managing a virtual bookshelf. Built using **Node.js** and **Hapi.js Framework**.

---

## Fitur Utama

- Tambah buku
- Lihat semua buku
- Lihat detail buku berdasarkan ID
- Ubah data buku
- Hapus buku
- Filter berdasarkan:
  - Nama buku (`?name=`)
  - Status baca (`?reading=`)
  - Status selesai dibaca (`?finished=`)
